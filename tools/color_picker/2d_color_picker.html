RGB Lerp Space Calculator

Provide three RGB values for a visualization of the 2D plane occupied by those points.

test

Motivation

I first created this with the goal of having a more systematic way of defining color palettes. It was created to try and choose better asset colors for a subterrenean-themed game I was working on. I also took inspiration from this tumblr post about a "flesh cloud" that artists could use to select skin tones more randomly; if you use the lerp tool on three distinct skin tones from the "flesh cloud", you can find the full flesh cloud as a subset of the returned 2D space.

Implementation

Lerp space calculator takes three colors and calculates the plane that linearly interpolates those points. It creates a 2D basis for this plane, and evaluates the resulting RGB value of each screen pixel, using pixel coordinates as coordinates in our new 2D basis.
